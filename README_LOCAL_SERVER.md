# 🎯 הדרכה מהירה - שרת מקומי

## 🚀 שלבים מהירים (Mac)

### 1️⃣ פתח Terminal

לחץ `Cmd+Space` → הקלד "Terminal" → Enter

### 2️⃣ נווט לתיקיית הפרויקט

```bash
cd /Users/amitzahy/Documents/Draft/FPL_25_26
```

### 3️⃣ הרץ את השרת

**אפשרות א' - הכי קלה**:
```bash
./START_SERVER.sh
```

**אפשרות ב' - ידנית**:
```bash
python3 -m http.server 8000
```

### 4️⃣ פתח דפדפן

גלוש ל: **http://localhost:8000**

### 5️⃣ לעצור את השרת

חזור ל-Terminal ולחץ `Ctrl+C`

## ⚠️ חשוב לדעת!

### ✅ מצב דמו
- לוחץ על "מצב דמו" ⇒ רואה את האתר עם נתונים אמיתיים
- **אין צורך בהתחברות** למטרות בדיקה!

### 🔄 רענון
אם עשית שינויים בקוד:
1. שמור את הקובץ
2. חזור לדפדפן
3. לחץ `Cmd+Shift+R` (רענון מלא)

### 🐛 אם משהו לא עובד
- בדוק ש-Terminal מציג: `Serving HTTP on :: port 8000`
- בדוק שהכתובת היא: `http://localhost:8000` (עם **http** ולא https)
- פתח Console (F12) לראות שגיאות

---

## 📱 בדיקה במובייל

1. הרץ שרת במחשב
2. מצא את ה-IP שלך:
   ```bash
   ifconfig | grep "inet " | grep -v 127.0.0.1
   ```
3. במובייל, גלוש ל: `http://[IP_ADDRESS]:8000`
