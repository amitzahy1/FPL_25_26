## 🎨 שינויי CSS

### Grid Layout
```css
.metrics-comparison-table {
    display: grid;
    grid-template-columns: repeat(2, 1fr);  /* 2 עמודות! */
    gap: 8px;
}

.metric-comparison-row {
    grid-template-columns: 130px 1fr;  /* קטן יותר */
    padding: 8px;  /* צפוף יותר */
}
```

### Fixtures Row (רוחב מלא)
```css
.fixtures-comparison-row {
    grid-column: 1 / -1;  /* תופס 2 עמודות */
}
```

### Value Boxes
```css
.metric-row-values {
    grid-template-columns: repeat(auto-fit, minmax(80px, 1fr));
    gap: 6px;
}

.metric-value-box {
    padding: 6px;
    border: 1px solid #e2e8f0;
}
```

### Responsive
```css
@media (max-width: 1024px) {
    .metrics-comparison-table {
        grid-template-columns: 1fr;  /* חזרה לעמודה 1 */
    }
}
```

## 🎯 Grid Layout - איך זה עובד?

### מבנה
```html
<div class="metrics-comparison-table">  ← Grid 2 cols
    <div class="metric-comparison-row">  ← Item 1
        <label>ציון דראפט</label>
        <values>95.2 | 94.8</values>
    </div>
    <div class="metric-comparison-row">  ← Item 2
        <label>העברות נטו</label>
        <values>+125 | +89</values>
    </div>
    ...
    <div class="fixtures-comparison-row">  ← Full width
        <label>משחקים קרובים</label>
        <fixtures>...</fixtures>
    </div>
</div>
```

### CSS Grid Magic
```css
/* Container */
.metrics-comparison-table {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    /* משמעות: 2 עמודות שוות */
}

/* Items */
.metric-comparison-row {
    /* כל item תופס תא אחד */
}

/* Exception: Fixtures takes full width */
.fixtures-comparison-row {
    grid-column: 1 / -1;
    /* משמעות: מעמודה 1 עד האחרונה */
}
```
